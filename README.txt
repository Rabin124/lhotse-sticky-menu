== Description ==

Lhotse Sticky Menu is a lightweight, simple  free WordPress plugin for sticky menu that allows you to design the menu on your website. The plugin is easy to use; even for the beginners. Lhotse Sticky Menu prevents your menu from disappearing when the user scrolls down the pages. Lhotse Sticky Menu sure has been designed to design the menu, however, you can make any element on your website sticky. You can make your website elements—Call to Action, your logo and more—sticky with Lhotse Sticky Menu. It comes with features like menu selector, background color, menu text color, Z index, opacity and font size. Additionally, you also have the option to whether or not display the sticky menu on the homepage only. Download Lhotse Sticky Menu today and highlight your menu by making them “sticky”.

== Installation ==

The easy way (via Dashboard) :

* Go to Plugins > Add New
* Type in the **Lhotse Sticky Menu** in Search Plugins box
* Click Install Now to install the plugin
* After Installation click activate to start using the **Lhotse Sticky Menu**
* Go to **Lhotse Sticky Menu** from Dashboard menu

Not so easy way (via FTP) :

* Download the **Lhotse Sticky Menu**
* Unarchive **Lhotse Sticky Menu** plugin
* Copy folder with lhotse-sticky-menu.zip
* Open the ftp \wp-content\plugins\
* Paste the plug-ins folder in the folder
* Go to admin panel => open item "Plugins" => activate **Lhotse Sticky Menu**
* Go to **Lhotse Sticky Menu** from Dashboard menu
